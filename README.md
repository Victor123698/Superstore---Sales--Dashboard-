# 📊 Superstore Sales Dashboard

## Overview
An interactive Sales Dashboard built in Power BI 
analyzing 75,000 sales transactions across 5 regions 
and 8 countries.

## 📸 Dashboard Preview
[![Dashboard](dashboard.png)]

## 📈 Key Metrics
| Metric | Value |
|---|---|
| Total Sales | $4.24M |
| Total Profit | $847K |
| Total Orders | 13,095 |
| Profit Margin | 20% |

## 🔍 Key Insights
- Technology drives highest profit
- Furniture has profitability issues
- Corporate customers most valuable
- North region most efficient
